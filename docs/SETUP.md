## Prerequisites

- Node.js 18+ and npm/yarn
- Git
- Supabase account
- Stripe account (for payments)

## Initial Setup

### 1. Clone Repository
```bash
git clone <repository-url>
cd Suppli
```

### 2. Environment Variables

See `IMPLEMENTATION_GUIDE.md` Phase 4.1 for detailed environment variable setup.

### 3. Install Dependencies

**Backend:**
```bash
cd server
npm install
```

**Frontend:**
```bash
cd frontend
npm install
```

## Development

### Backend
```bash
cd server
npm run dev
```

### Frontend
```bash
cd frontend
npm run dev
```
